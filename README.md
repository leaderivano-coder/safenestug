Problem & Solution
Problem: Communities in Uganda face challenges in reporting child abuse cases due to limited access, weak linkages with formal structures, and lack of anonymity.
Solution: SafeNest provides a secure, community‑based reporting platform with role‑based access, multimedia evidence submission, and integration with national child protection stakeholders (MGLSD, UPF, UNICEF, NGOs).

🔹 Setup Instructions
Install XAMPP/WAMP and create a MySQL database safenest_db.

Import the provided SQL schema (safenest.sql).

Copy project files into the server root (htdocs/safenest).

Edit config.php with database credentials.

Start Apache/MySQL → open http://localhost/safenest in your browser.
